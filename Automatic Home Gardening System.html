Automatic Home Gardening System

  • This project was designed to be an easy to use home gardening system with an interactive user interface. Device can be used to irrigate multiple plants with various amounts of water on separate individually set times.
  • The project uses an LCD display and push buttons as a user interface. The water timing for each plant can be set similar to that of an alarm . The irrigation is controlled through a DC pump where each plant is attached with an individual unit.Time is kept by the microprocessor using a RTC module
  • Since the flow of water through each motor is constant, the volume of water is controlled by setting the time for which each motor is switched on.
  • An overhead reservoir is required to hold the adequate amount the water.
